# Okko

## 1. Тема и целевая аудитория

### 1.1 Тема
**Okko** — российский мультиплатформенный онлайн-кинотеатр, входящий в экосистему Сбера. 
Сервис позволяет:
- Смотреть фильмы, сериалы, мультфильмы и ТВ-каналы по подписке
- Получать эксклюзивный доступ к оригинальным проектам Okko
- Смотреть спортивные трансляции
- Скачивать контент для просмотра офлайн
- Использовать рекомендательную систему на основе машинного обучения

---

### 1.2 Функционал MVP
Минимально жизнеспособный продукт (MVP) включает в себя:
- Регистрация и авторизация (по номеру телефона или через соцсети)
- Поиск контента
- Просмотр спортивных трансляций
- Рекомендательная система 
- Видеоплеер с регулировкой качества и автоматически сгенерированными субтитрами
- Сохранение контента в «Смотреть позже»
- История просмотров
