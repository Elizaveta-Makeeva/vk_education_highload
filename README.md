# Okko

## 1. Тема и целевая аудитория

### 1.1 Тема
**Okko** — российский мультиплатформенный онлайн-кинотеатр, входящий в экосистему Сбера. 
Сервис позволяет:
- Смотреть фильмы, сериалы, мультфильмы и ТВ-каналы по подписке
- Получать эксклюзивный доступ к оригинальным проектам Okko
- Смотреть спортивные трансляции
- Скачивать контент для просмотра офлайн
- Использовать рекомендательную систему на основе машинного обучения


### 1.2 Функционал MVP
Минимально жизнеспособный продукт (MVP) включает в себя:
- Регистрация и авторизация (по номеру телефона или через соцсети)
- Поиск контента
- Просмотр спортивных трансляций
- Рекомендательная система 
- Видеоплеер с регулировкой качества и автоматически сгенерированными субтитрами
- Сохранение контента в «Смотреть позже»
- История просмотров


### 1.3 Целевая аудитория

#### География - Российская Федерация и страны СНГ

Размер аудитории:

Месячный охват — 17 млн человек

Дневной охват — 5 млн человек

#### Социально-демографические характеристики

- **Средний возраст:** 16–55 лет
- **Гендерное распределение:**
  - Женщины: **50%**
  - Мужчины: **50%**

## 3. Глобальная балансировка нагрузки

### 3.1 Функциональное разбиение по доменам 

* okko.tv - Главный домен для пользователей, каталог, страницы контента.
* kids.okko.tv - Отдельный поддомен для контента, ориентированного на детей.
* about.okko.tv - О компании, вакансии, пресс-центр.
* blog.okko.tv - Новости сервиса, статьи о контенте, анонсы.
* merch.okko.tv - Магазин с товарами по мотивам фильмов и сериалов.
* okko.sport - Отдельный домен для спортивных трансляций
* auth.okko.tv - Сервис аутентификации
* auth.okko.sport - Сервис аутентификации для спортивного портала
* clients-static.okko.tv - CDN для статики клиентских приложений
* player-static.okko.tv - CDN для плеера
* static.okko.tv - Основной CDN для медиа-контента.
* stat.okko.tv - Сервис сбора статистики и аналитики.
* stat.okko.sport - Сервис сбора статистики для спортивного портала.

### 3.2 Расположение ДЦ

Okko имеет два дата-центра: в Москве и Санкт-Петербурге [[6](https://thecode.media/zapusk-kino/amp/#:~:text=%D0%A3%20%D0%BD%D0%B0%D1%81%20%D0%B4%D0%B2%D0%B0%20%D0%B4%D0%B0%D1%82%D0%B0%2D%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%3A%20%D0%B2%20%D0%9F%D0%B8%D1%82%D0%B5%D1%80%D0%B5%20%D0%B8%20%D0%B2%20%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B5)]. При размещении ДЦ таким образом перекрываются два крупнейших транспортных узла связи России: Москва — центр хранения контента и столица, Санкт-Петербург — ключевой узел, через который проходят крупнейшие магистральные каналы связи на Северо-Запад и в Европу.

### 3.4 Балансировка и механизм регулировки трафика между ДЦ

При резких всплесках трафика (в 3-5 раз выше обычного, как во время Олимпиады) часть backend-сервисов Okko динамически переносится в частное облако Cloud.ru [[8](cloud.cnews.ru/news/line/2026-02-27_okko_ispolzoval_oblachnye#comments)]. К двум физическим ДЦ добавляется облачный ДЦ. Балансировщик нагрузки перенаправляет часть запросов не в перегруженный ДЦ, а в облачный кластер, который может быть развернут в любом регионе и масштабироваться по запросу.
